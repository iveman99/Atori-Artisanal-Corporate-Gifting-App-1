Atori is a modern **Corporate Gifting App** that helps businesses explore and select curated gifts for clients, partners, and employees. The app provides a seamless way to browse, customize, and track corporate gifts, aligning with Atori’s vision of strengthening professional relationships through thoughtful gifting.  

This project showcases a **complete Android app workflow**: front-end design (XML, Kotlin), back-end integration (Firebase), and cloud database management (Firestore, Realtime Database).

## 🔎 Project Purpose  
The Atori app was designed to simplify and modernize corporate gifting.  

It answers key business needs like:  
- How can businesses browse and select gifts quickly?  
- How to track orders efficiently from one platform?  
- How can admins manage products, users, and orders easily?  
- How to provide a **secure, scalable** solution for both clients and admins?  

This app is useful for **corporates, HR managers, marketing teams, and gifting companies** looking to digitalize their gifting process.

## 🛠 Tech Stack  
The app was developed using the following technologies:  

- 📱 **Android (Kotlin, XML)** – UI/UX and business logic  
- ☁️ **Firebase Authentication** – secure login & registration  
- 📂 **Firebase Realtime Database & Firestore** – storing users, products, and orders  
- 📊 **Firebase Console** – admin monitoring & analytics  
- 🖼 **Glide** – for image loading in the client app  
- 🔔 **Intents & Fragments** – navigation and communication

## 📑 Features  

### 👥 Client App  
- 🔐 Login & Registration (Firebase Auth)  
- 🎁 Browse curated gift items  
- 📝 Gift customization options  
- 📞 Order request via call (no direct online order)  
- 📷 Showcase section (videos/images for promotions)  
- 🌙 Modern, user-friendly interface

### 🛠 Admin App  
- 🔐 Login & Register (Firebase Auth)  
- 📊 Dashboard – revenue & order stats  
- ➕ Add / Update / Delete products (Name, Rating, Description, Blogger image/video URL)  
- 📋 Orders list with status update  
- 👤 User list (Name & Email of cart users)  
- 🎨 Theme consistent with client app
